///////////////////////////////////////////////////////////////////////////////
컴파일을 위한 선행 조치
///////////////////////////////////////////////////////////////////////////////

1. MS Flatform SDK 설치해야 함

http://www.microsoft.com/msdownload/platformsdk/sdkupdate/

2. Visual Studio 에서 Platform SDK include 디렉토리 설정

도구 -> 옵션 -> 프로젝트 -> VC++ 디렉토리

3. 프로젝트 속성 -> C/C++ -> 전처리기 정의에 아래 전처리기 추가.

_WIN32_WINNT=0x0500;

4. 프로젝트 속성 -> 코드생성 -> 런타임 라이브러리

다중 스레드 디버그(/MTd)
